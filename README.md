# **TryHackMe Writeups** 
It's a repository contains my structured notes, write-ups, and documentation from various rooms and learning paths completed on TryHackMe. It serves as a personal knowledge base for cybersecurity topics including penetration testing, networking, Linux/Windows fundamentals, Active Directory, and web exploitation.

Each entry is focused on practical learning, including commands used, methodologies, tools, and key takeaways from hands-on labs. The goal is to track progress, reinforce understanding, and build a solid reference for real-world cybersecurity skills development.

This repository is continuously updated as new rooms are completed and new concepts are explored.
